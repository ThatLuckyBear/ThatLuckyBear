## How to Use
This document will be a disorganized list of ideas for VGATWD essays. Each idea should have:
- a title
- game concept
- life concept
- brief explanation of how they tie together

When the explanation can't stay brief it's time to make an essay document in the /VGATWD-Essays/ directory.

Once an idea has an essay doc in /VGATWD-Essays/ a link to that file should be added to the listing in the ideas doc (this one).
